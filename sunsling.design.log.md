# About
This is the main way of governing design work for the /dpdp project.
Design work must be governed in another way than project work. Instead of building only a backlog, and project plans, design work can have a backlog also but is mainly driven by judgement and reflection, and this is where a log-file is more in line with how we work.

## Graphical view
There is also a complementary view which is also useful for design work, in terms of 'spaces' of work, but this is not yet in place for this project.
The file in this case will be called 'sunsling.design.spaces.drawio'
